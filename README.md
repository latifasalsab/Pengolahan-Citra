- NumPy
- Matplotlib
- Scikit-image

## ⚙️ Setup
1. Clone repository ini
```bash
   git clone https://github.com/username/pengolahan-citra.git
```
2. Buat virtual environment
```bash
   python -m venv venv
   venv\Scripts\activate
```
3. Install dependencies
```bash
   pip install scikit-image matplotlib numpy ipykernel
```
4. Buka file `.ipynb` di VSCode dan pilih kernel `venv`
